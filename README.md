Power BI E-Commerce Sales Analysis Dashboard

Interactive dashboard built to track and analyze online sales data using complex parameters, slicers, table joins, and custom calculations for drill-down insights.
​
Project Learnings: 
Created interactive dashboard to track and analyze online sales data
Used complex parameters and slicers for worksheet drill-down
Created connections, joined new tables, calculations to manipulate data
Used different user-driven custom visualizations (bar chart, pie chart, donut chart, clustered bar chart, scatter chart, line chart, area chart, map, slicers, etc.)

Features:
Data Modeling: Table relationships and custom calculations
Interactivity: Complex parameters, slicers, and drill-down filters
Visualizations: Bar, pie, donut, clustered bar, scatter, line, area charts, maps, slicers
User-Driven: Custom filters enabling data exploration

Setup:
Download ecommerce-sales-dashboard.pbix
Open in Power BI Desktop
Refresh data > Use slicers for interactive analysis
Publish to Power BI Service for sharing
